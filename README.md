# LTSラマン校正トムソン散乱解析アプリ

LightSpeed/LightFieldなどから出力した `.spe` スペクトルを読み込み、ラマン散乱による絶対値校正を用いて、トムソン散乱スペクトルから電子密度 `ne` と電子温度 `Te` を求めるための試作アプリです。

起動するときは、このフォルダ内の次のファイルをダブルクリックしてください。

```text
起動.bat
```

アプリを閉じると、入力欄の値は同じフォルダ内の `app_settings.json` に自動保存されます。次回起動時には、最後に閉じたときの値が復元されます。

## 1. ラマン校正TS解析タブ

このタブでは、トムソン散乱 `.spe` とラマン散乱 `.spe` を入力し、電子密度・電子温度を計算します。
画面に収まりきらない場合は、右側または下側のスクロールバーで移動してください。

### ファイル入力

| 入力欄 | 入れる値 | 説明 |
|---|---|---|
| トムソンSPE | トムソン散乱スペクトルの `.spe` ファイル | 電子温度・電子密度を求めたい測定データを指定します。 |
| ラマンSPE | 校正用ラマン散乱スペクトルの `.spe` ファイル | 単一ラマン校正で用いるラマンデータを指定します。 |
| 保存先 | 解析結果を保存するフォルダ | `latest_lts_result.csv` などの結果CSVが保存されます。 |

### スペクトル位置・ROI

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| TS y最小 | pixel | トムソン信号を縦方向に積算する開始行 | 2次元CCD画像で、トムソン信号が写っている範囲の下端または上端を指定します。 |
| TS y最大 | pixel | トムソン信号を縦方向に積算する終了行 | `TS y最小 < TS y最大` となるようにします。例: `600` から `970`。 |
| レーザー中心 | pixel | レーザー波長またはレイリー線の中心ピクセル | トムソン散乱フィットの中心位置として使います。例: `536.8749`。 |
| ラマン中心 | pixel | ラマンピーク中心ピクセル | 分かっている場合は入力します。自動検出したい場合は空欄でも構いません。 |

### トムソンスペクトルのフィット範囲・背景範囲

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| TS fit最小 | pixel | トムソンピークをフィットする範囲の開始pixel | ピーク本体が入る範囲にします。初期値は `360`。 |
| TS fit最大 | pixel | トムソンピークをフィットする範囲の終了pixel | `TS fit最小 < TS fit最大` にしてください。初期値は `760`。 |
| 背景左最小 | pixel | トムソンピーク左側のベースライン範囲の開始pixel | ピークから外れた左側の平坦な領域を指定します。初期値は `180`。 |
| 背景左最大 | pixel | トムソンピーク左側のベースライン範囲の終了pixel | `背景左最小 < 背景左最大` にしてください。初期値は `330`。 |
| 背景右最小 | pixel | トムソンピーク右側のベースライン範囲の開始pixel | ピークから外れた右側の平坦な領域を指定します。初期値は `800`。 |
| 背景右最大 | pixel | トムソンピーク右側のベースライン範囲の終了pixel | `背景右最小 < 背景右最大` にしてください。初期値は `1000`。 |
| 平滑化幅 | pixel | トムソンフィット前の移動中央値フィルタ幅 | ノイズが強い場合に大きくします。初期値は `21`。 |
| fitしきい値 | - | フィットに使う点を決める相対しきい値 | ピーク最大値に対する割合です。初期値は `0.15`。 |
| 逆スリット最小 | pixel | フィットから除外する逆スリット欠損範囲の開始pixel | 532 nm付近など、信号が切れている範囲を指定します。使わない場合は空欄。 |
| 逆スリット最大 | pixel | フィットから除外する逆スリット欠損範囲の終了pixel | `逆スリット最小 < 逆スリット最大` にしてください。使わない場合は空欄。 |

背景左範囲と背景右範囲は、同じ位置にしないでください。左右範囲の代表x座標が一致すると、ベースライン直線の傾き計算で分母が0になるため、解析を止めます。
トムソンスペクトルのガウスフィットは、ベースライン差し引き後に非線形最小二乗法で行います。逆スリット範囲を入力した場合、そのpixel範囲は残差計算から除外されます。

### 波長・幾何条件

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| 波長校正 | nm/pixel | 分光器の波長校正値 | 1 pixel が何 nm に相当するかを入力します。現在の目安は `0.021`。電子温度計算に直接効きます。 |
| レーザー波長 | nm | プローブレーザーの波長 | Nd:YAG第2高調波なら `532`。 |
| 散乱角 | deg | 入射レーザーと観測方向のなす角 | 90度配置なら `90`。散乱波数と電子温度換算に使います。 |
| 装置幅sigma | pixel | 装置関数の標準偏差 | レイリー線などから求めた装置幅を入れます。補正しない場合は `0`。 |

### ラマン校正用の物理量

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| 水素圧力 | Pa | ラマン散乱を測定したときのチャンバー内水素圧力 | 気体分子数密度 `n_gas = P / (k_B T)` の計算に使います。 |
| 気体温度 | K | ラマン散乱測定時の気体温度 | 室温なら `300` を目安にします。 |
| ラマン有効断面積 | m^2 | 校正に使うラマン散乱の有効断面積 | 論文で窒素Stokes線に用いられていた値の例は `3.82e-34`。水素ラマンを使う場合は、水素の条件に合う値へ変更してください。 |
| TS断面積 | m^2 | トムソン散乱の有効断面積 | 論文式に合わせる場合の初期値は古典電子半径の二乗 `7.94e-30`。 |
| ラマンshot数 | shots | ラマン散乱スペクトルの積算ショット数 | 1ショットなら `1`。複数ショット積算ならその数を入れます。 |
| TS shot数 | shots | トムソン散乱スペクトルの積算ショット数 | 1ショットなら `1`。 |
| ラマンEnergy | mJ | ラマン測定時のレーザーエネルギー | レーザーエネルギー補正をしない場合は `1`。 |
| TS Energy | mJ | トムソン測定時のレーザーエネルギー | ラマンと同じく、補正しない場合は `1`。 |
| 補正係数 | - | 透過率・偏光・光学系差などをまとめた追加補正係数 | 追加補正がない場合は `1`。既知の光学補正がある場合だけ変更します。 |

規格化には次を使います。

```text
ラマン規格化信号 = ラマン面積 / (ラマンshot数 * ラマンEnergy)
TS規格化信号     = TS面積     / (TS shot数     * TS Energy)
```

### 複数Stokesピーク合計

ラマン散乱スペクトルに複数のStokesピークがあり、その合計強度で校正したい場合に使います。

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| Stokes探索最小 | pixel | Stokesピークを探す範囲の最小ピクセル | 解析したいラマンピーク群が入る範囲を指定します。 |
| Stokes探索最大 | pixel | Stokesピークを探す範囲の最大ピクセル | 探索範囲を広げすぎると迷光やノイズを拾うため、必要範囲に絞ります。 |
| ラマンy最小 | pixel | ラマンスペクトルを縦方向に積算する開始行 | SPEビューアの赤枠ROIから反映できます。空欄なら全y範囲を使います。 |
| ラマンy最大 | pixel | ラマンスペクトルを縦方向に積算する終了行 | `ラマンy最小 < ラマンy最大` にしてください。 |
| Stokesピーク数 | 個 | 検出するStokesピーク数 | SPEビューアの `Stokesピーク数` から反映できます。 |
| ピークしきい値 | - | ピーク検出の相対しきい値 | 最大強度に対する割合です。例: `0.08` は最大値の8%以上を候補にします。 |

ラマンフィットは常に複数Stokesピークの合計で行います。各ピークは非線形最小二乗ガウスフィットで評価し、`ラマン有効断面積` には、合計したStokesピーク群に対応する断面積を入力してください。

### 測定日内ドリフト補間

測定の最初と最後にラマンを測定し、その間にトムソン散乱を測定した場合に使います。ラマン校正係数を時間で線形補間します。

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| TS時刻 | h | トムソン散乱測定時刻 | 例: 15時30分なら `15.5`。 |
| 開始時刻 | h | 開始側ラマン測定時刻 | 例: `14.0`。 |
| 終了時刻 | h | 終了側ラマン測定時刻 | 例: `17.0`。 |
| 開始圧力 | Pa | 開始側ラマン測定時の水素圧力 | 開始ラマンの気体密度計算に使います。 |
| 終了圧力 | Pa | 終了側ラマン測定時の水素圧力 | 終了ラマンの気体密度計算に使います。 |
| 開始shot数 | shots | 開始側ラマンの積算ショット数 | 1ショットなら `1`。 |
| 終了shot数 | shots | 終了側ラマンの積算ショット数 | 1ショットなら `1`。 |
| 開始Energy | mJ | 開始側ラマンのレーザーエネルギー | 補正しない場合は `1`。 |
| 終了Energy | mJ | 終了側ラマンのレーザーエネルギー | 補正しない場合は `1`。 |

補間は次の形で行います。

```text
k(t) = k_start + (k_end - k_start) * (t - t_start) / (t_end - t_start)
```

### 選択ボタン

| 選択欄 | 選ぶ内容 | 説明 |
|---|---|---|
| 校正 | 単一ラマン / 時間補間 | ラマン1点で校正するか、開始・終了ラマンでドリフト補間するかを選びます。 |
| ラマン面積 | 複数Stokesピーク合計で固定 | SPEビューアで欠けていない範囲を確認し、必要なピーク範囲とピーク数を指定してください。 |

## 2. フィット表示タブ

`トムソンフィット` タブには、校正タブで解析したトムソンスペクトル、非線形最小二乗ガウスフィット曲線、残差が表示されます。逆スリット範囲を指定している場合、その範囲はフィットの残差計算から除外されます。

`ラマンフィット` タブには、校正タブで解析したラマンスペクトル、複数Stokesピークの合計フィット曲線、残差が表示されます。ラマンは単一ピークではなく、指定されたStokesピーク数の合計面積で校正します。

## 3. ラマン圧力依存性タブ

このタブでは、複数のラマン散乱スペクトルを読み込み、圧力に対してピーク強度が比例しているか確認します。

### 入力欄

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| フォルダ | - | ラマン散乱 `.spe` が保存されているフォルダ | 参照ボタンでフォルダを選びます。 |
| 圧力 | Pa | 各 `.spe` ファイルを測定したときの水素圧力 | ファイルごとに入力します。横軸の値になります。 |
| ラマン中心 | pixel | ラマンピーク中心ピクセル | 分かっている場合は入力します。空欄なら自動検出します。 |
| 縦軸 | counts または count*pixel | ピーク高さ / ガウス面積 | ピーク高さは最大強度、ガウス面積はフィットから得た積分強度です。 |

解析後、横軸を圧力、縦軸をピーク強度または面積としたグラフを表示します。結果は `raman_pressure_dependence.csv` に保存されます。

## 4. 出力値の意味

| 表示値 | 単位 | 意味 |
|---|---:|---|
| `n_gas` | m^-3 | 入力した圧力・温度から計算した中性気体分子数密度です。 |
| `k` | m または m/mJ相当 | ラマン散乱から求めた装置感度を含む校正係数です。 |
| `ne` | m^-3 | トムソン散乱から求めた電子密度です。 |
| `ne` | cm^-3 | `m^-3` を `cm^-3` に換算した電子密度です。 |
| `Te` | eV | トムソン散乱スペクトル幅から求めた電子温度です。 |
| `alpha` | - | 散乱パラメータです。1に近い、または1を超える場合、単純な非集団的ガウス近似の妥当性に注意が必要です。 |
| `R^2` | - | フィッティングの決定係数です。1に近いほど、データをよく説明しています。 |

トムソン解析では `latest_thomson_fit_curve.csv` も保存されます。このCSVには、各pixelの生スペクトル、平滑化スペクトル、背景、フィット曲線、残差が入ります。R^2が悪い場合は、残差が片側だけ大きい、裾だけ合わない、中心付近だけ外れる、などを確認してください。

## 5. スペクトル差し引きタブ

このタブでは、4種類の `.spe` スペクトル画像からトムソン信号画像を作成し、校正タブで使える `.spe` として保存します。

### 入力欄

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| 保存先 | - | トムソン信号ファイルを保存するフォルダ | 校正タブで読み込む前処理済みSPEを置きたいフォルダを指定します。 |
| CSV保存 | - | チェックすると2D画像CSVも保存 | 出力名は `thomson_signal.csv` です。行がCCDのy方向、列がx方向です。 |
| SPEは常に保存 | - | 表示のみ | `.spe` は常に `thomson_signal.spe` として保存されます。差分で負値が出るため、データ部はfloat32形式で保存します。 |
| 全信号SPE | - | プローブあり・プラズマありで取得した `.spe` | 全信号、プラズマ自発光、プローブ迷光、背景光を含む画像です。 |
| プラズマ自発光SPE | - | プローブなし・プラズマありで取得した `.spe` | プラズマの自発光成分を差し引くために使います。 |
| プローブ迷光SPE | - | プローブあり・プラズマなしで取得した `.spe` | プローブ光由来の迷光成分を差し引くために使います。 |
| 背景光SPE | - | プローブなし・プラズマなしで取得した `.spe` | 検出器背景や環境光など、二重に差し引かれる背景成分を戻すために加算します。 |

計算は次の形で行います。

```text
トムソン信号 = 全信号 - プラズマ自発光 - プローブ迷光 + 背景光
```

4つのSPEは、CCD画像サイズが同じである必要があります。サイズが異なる場合は保存せずエラーになります。

### 表示機能

| ボタン・表示 | 内容 | 説明 |
|---|---|---|
| プレビュー更新 | トムソン信号画像と横方向スペクトルを表示 | 保存せずに、4つのSPEから計算される信号を確認します。 |
| トムソン信号保存 | トムソン信号画像と横方向スペクトルを表示し、SPE/CSVを保存 | `.spe` は常に保存され、CSVはチェック時のみ保存されます。 |
| 差し引き後プレビュー | 左に2D画像、右に横方向積算スペクトルを表示 | 画像表示は明るさを自動調整しています。保存される数値データ自体は変更されません。 |

### スペクトルフィッティング

`トムソンガウスフィット` ボタンを押すと、4つのSPEから計算したトムソン信号画像を横方向に積算し、自動で最小二乗ガウスフィットします。

フィットは `counts = A exp[-(x-x0)^2/(2 sigma^2)] + B` の形で行います。中心 `x0` と幅 `sigma` の候補を探索し、それぞれの候補で振幅 `A` と背景 `B` を線形最小二乗で求め、残差平方和が最小の結果を採用します。フィット後は左にデータとフィット曲線、右に残差が表示されます。表示が小さい場合は、タブ右側・下側のスクロールバーで移動してください。

## 6. SPEビューアタブ

このタブでは、任意の `.spe` ファイルを読み込み、CCD上の2D画像と横方向積算スペクトルを表示します。

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| SPEファイル | - | 表示したい `.spe` ファイル | 通常の測定SPE、差し引き後SPEのどちらも読み込めます。 |
| x最小 | pixel | スペクトル表示に使う横方向範囲の開始pixel | 空欄なら画像の左端から使います。 |
| x最大 | pixel | スペクトル表示に使う横方向範囲の終了pixel | 空欄なら画像の右端まで使います。`x最小 < x最大` にしてください。 |
| y最小 | pixel | スペクトル化するときの縦方向積算開始行 | 空欄なら画像の一番上から積算します。 |
| y最大 | pixel | スペクトル化するときの縦方向積算終了行 | 空欄なら画像の一番下まで積算します。`y最小 < y最大` にしてください。 |
| Stokesピーク数 | 個 | ラマン校正で合計するStokesピーク数 | `解析タブへ反映` ボタンで校正タブの `Stokesピーク数` に反映されます。 |

`表示` ボタンを押すと、左側にSPE画像と指定ROI、右側に指定ROI内でy方向積算したスペクトルが表示されます。画像上にはx/y pixel値とROI範囲が表示され、スペクトルのx軸も実際のpixel値になります。画像上でマウスをドラッグすると、赤枠ROIを任意の縦横範囲で指定できます。

`解析タブへ反映` ボタンを押すと、ビューアで指定したROIが解析タブへ反映されます。y範囲は `TS y最小/最大` と `ラマンy最小/最大`、x範囲はラマンの `Stokes探索最小/最大` と `TS fit最小/最大` に入ります。`Stokesピーク数` を入力している場合は、その値も校正タブへ反映されます。

## 7. 入力に迷ったときの初期値

| 項目 | 初期値の目安 |
|---|---:|

| 補正係数 | 補正しない場合は `1` |

| 背景左範囲 | 初期値は `180` から `330` pixel |

| SPEビューア x/y範囲 | 全体を積算するなら空欄 |

## 8. 注意

- 複数Stokesピークを合計する場合、入力するラマン有効断面積も合計ピークに対応した値にしてください。
- 波長校正、レーザー波長、水素圧力、気体温度、ラマン有効断面積、TS断面積、shot数、Energy、補正係数は0にしないでください。0または負値の場合、密度校正の分母が0になるため解析を止めます。
- 校正タブでエラーが出た場合は、ログ欄に `停止ステップ`、直前の入力値・中間値、内部トレース末尾が表示されます。エラー原因の切り分けには、そのログを確認してください。
- 現在の電子温度計算は、トムソン散乱スペクトルをガウス形で近似する試作実装です。集団的散乱が強い条件では、より厳密なスペクトル形状モデルが必要です。
- `R^2` が高くても、迷光・飽和・背景処理の影響が残る場合があります。必ずフィット図も確認してください。
- 差し引き後SPEはfloat32形式で保存します。読み込み側ソフトがfloat32 SPEに対応していない場合は、CSVを使って確認してください。
- 入力値を初期状態に戻したい場合は、アプリを閉じた状態で `app_settings.json` を削除してください。次回起動時に標準の初期値へ戻ります。
