# LTSラマン校正トムソン散乱解析アプリ

LightSpeed/LightFieldなどから出力した `.spe` スペクトルを読み込み、ラマン散乱による絶対値校正を用いて、トムソン散乱スペクトルから電子密度 `ne` と電子温度 `Te` を求めるための試作アプリです。

起動するときは、このフォルダ内の次のファイルをダブルクリックしてください。

```text
起動.bat
```

## 1. ラマン校正TS解析タブ

このタブでは、トムソン散乱 `.spe` とラマン散乱 `.spe` を入力し、電子密度・電子温度を計算します。

### ファイル入力

| 入力欄 | 入れる値 | 説明 |
|---|---|---|
| トムソンSPE | トムソン散乱スペクトルの `.spe` ファイル | 電子温度・電子密度を求めたい測定データを指定します。 |
| ラマンSPE | 校正用ラマン散乱スペクトルの `.spe` ファイル | 単一ラマン校正で用いるラマンデータを指定します。 |
| 保存先 | 解析結果を保存するフォルダ | `latest_lts_result.csv` などの結果CSVが保存されます。 |

### スペクトル位置・ROI

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| TS y最小 | pixel | トムソン信号を縦方向に積算する開始行 | 2次元CCD画像で、トムソン信号が写っている範囲の下端または上端を指定します。 |
| TS y最大 | pixel | トムソン信号を縦方向に積算する終了行 | `TS y最小 < TS y最大` となるようにします。例: `600` から `970`。 |
| レーザー中心 | pixel | レーザー波長またはレイリー線の中心ピクセル | トムソン散乱フィットの中心位置として使います。例: `536.8749`。 |
| ラマン中心 | pixel | ラマンピーク中心ピクセル | 分かっている場合は入力します。自動検出したい場合は空欄でも構いません。 |
| ストップ最小 | pixel | 波長ストップで隠れた範囲の最小ピクセル | ラマンピークの一部が遮られている場合、その範囲をフィットから除外します。ない場合は空欄にします。 |
| ストップ最大 | pixel | 波長ストップで隠れた範囲の最大ピクセル | ストップ最小とセットで使います。ない場合は空欄にします。 |

### 波長・幾何条件

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| 波長校正 | nm/pixel | 分光器の波長校正値 | 1 pixel が何 nm に相当するかを入力します。現在の目安は `0.021`。電子温度計算に直接効きます。 |
| レーザー波長 | nm | プローブレーザーの波長 | Nd:YAG第2高調波なら `532`。 |
| 散乱角 | deg | 入射レーザーと観測方向のなす角 | 90度配置なら `90`。散乱波数と電子温度換算に使います。 |
| 装置幅sigma | pixel | 装置関数の標準偏差 | レイリー線などから求めた装置幅を入れます。補正しない場合は `0`。 |

### ラマン校正用の物理量

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| 水素圧力 | Pa | ラマン散乱を測定したときのチャンバー内水素圧力 | 気体分子数密度 `n_gas = P / (k_B T)` の計算に使います。 |
| 気体温度 | K | ラマン散乱測定時の気体温度 | 室温なら `300` を目安にします。 |
| ラマン有効断面積 | m^2 | 校正に使うラマン散乱の有効断面積 | 論文で窒素Stokes線に用いられていた値の例は `3.82e-34`。水素ラマンを使う場合は、水素の条件に合う値へ変更してください。 |
| TS断面積 | m^2 | トムソン散乱の有効断面積 | 論文式に合わせる場合の初期値は古典電子半径の二乗 `7.94e-30`。 |
| ラマンshot数 | shots | ラマン散乱スペクトルの積算ショット数 | 1ショットなら `1`。複数ショット積算ならその数を入れます。 |
| TS shot数 | shots | トムソン散乱スペクトルの積算ショット数 | 1ショットなら `1`。 |
| ラマンEnergy | mJ | ラマン測定時のレーザーエネルギー | レーザーエネルギー補正をしない場合は `1`。 |
| TS Energy | mJ | トムソン測定時のレーザーエネルギー | ラマンと同じく、補正しない場合は `1`。 |
| 補正係数 | - | 透過率・偏光・光学系差などをまとめた追加補正係数 | 追加補正がない場合は `1`。既知の光学補正がある場合だけ変更します。 |

規格化には次を使います。

```text
ラマン規格化信号 = ラマン面積 / (ラマンshot数 * ラマンEnergy)
TS規格化信号     = TS面積     / (TS shot数     * TS Energy)
```

### 複数Stokesピーク合計

ラマン散乱スペクトルに複数のStokesピークがあり、その合計強度で校正したい場合に使います。

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| Stokes探索最小 | pixel | Stokesピークを探す範囲の最小ピクセル | 解析したいラマンピーク群が入る範囲を指定します。 |
| Stokes探索最大 | pixel | Stokesピークを探す範囲の最大ピクセル | 探索範囲を広げすぎると迷光やノイズを拾うため、必要範囲に絞ります。 |
| 最大ピーク数 | 個 | 検出するピークの最大数 | 例: `12`。実際に使うStokesピーク数より少し多めにします。 |
| ピークしきい値 | - | ピーク検出の相対しきい値 | 最大強度に対する割合です。例: `0.08` は最大値の8%以上を候補にします。 |

この方法を使う場合、`ラマン有効断面積` には、合計したStokesピーク群に対応する断面積を入力してください。

### 測定日内ドリフト補間

測定の最初と最後にラマンを測定し、その間にトムソン散乱を測定した場合に使います。ラマン校正係数を時間で線形補間します。

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| TS時刻 | h | トムソン散乱測定時刻 | 例: 15時30分なら `15.5`。 |
| 開始時刻 | h | 開始側ラマン測定時刻 | 例: `14.0`。 |
| 終了時刻 | h | 終了側ラマン測定時刻 | 例: `17.0`。 |
| 開始圧力 | Pa | 開始側ラマン測定時の水素圧力 | 開始ラマンの気体密度計算に使います。 |
| 終了圧力 | Pa | 終了側ラマン測定時の水素圧力 | 終了ラマンの気体密度計算に使います。 |
| 開始shot数 | shots | 開始側ラマンの積算ショット数 | 1ショットなら `1`。 |
| 終了shot数 | shots | 終了側ラマンの積算ショット数 | 1ショットなら `1`。 |
| 開始Energy | mJ | 開始側ラマンのレーザーエネルギー | 補正しない場合は `1`。 |
| 終了Energy | mJ | 終了側ラマンのレーザーエネルギー | 補正しない場合は `1`。 |

補間は次の形で行います。

```text
k(t) = k_start + (k_end - k_start) * (t - t_start) / (t_end - t_start)
```

### 選択ボタン

| 選択欄 | 選ぶ内容 | 説明 |
|---|---|---|
| 校正 | 単一ラマン / 時間補間 | ラマン1点で校正するか、開始・終了ラマンでドリフト補間するかを選びます。 |
| 面積 | ガウス面積 / 直接積分 / 複数Stokes合計 | ラマン強度の評価方法を選びます。波長ストップでピークが欠ける場合は、基本的にガウス面積を使います。 |

## 2. ラマン圧力依存性タブ

このタブでは、複数のラマン散乱スペクトルを読み込み、圧力に対してピーク強度が比例しているか確認します。

### 入力欄

| 入力欄 | 単位 | 入れる値 | 目安・説明 |
|---|---:|---|---|
| フォルダ | - | ラマン散乱 `.spe` が保存されているフォルダ | 参照ボタンでフォルダを選びます。 |
| 圧力 | Pa | 各 `.spe` ファイルを測定したときの水素圧力 | ファイルごとに入力します。横軸の値になります。 |
| ラマン中心 | pixel | ラマンピーク中心ピクセル | 分かっている場合は入力します。空欄なら自動検出します。 |
| ストップ最小 | pixel | 波長ストップで隠れた範囲の最小ピクセル | ピーク欠損部をフィットから除外します。ない場合は空欄。 |
| ストップ最大 | pixel | 波長ストップで隠れた範囲の最大ピクセル | ストップ最小とセットで使います。ない場合は空欄。 |
| 縦軸 | counts または count*pixel | ピーク高さ / ガウス面積 | ピーク高さは最大強度、ガウス面積はフィットから得た積分強度です。 |

解析後、横軸を圧力、縦軸をピーク強度または面積としたグラフを表示します。結果は `raman_pressure_dependence.csv` に保存されます。

## 3. 出力値の意味

| 表示値 | 単位 | 意味 |
|---|---:|---|
| `n_gas` | m^-3 | 入力した圧力・温度から計算した中性気体分子数密度です。 |
| `k` | m または m/mJ相当 | ラマン散乱から求めた装置感度を含む校正係数です。 |
| `ne` | m^-3 | トムソン散乱から求めた電子密度です。 |
| `ne` | cm^-3 | `m^-3` を `cm^-3` に換算した電子密度です。 |
| `Te` | eV | トムソン散乱スペクトル幅から求めた電子温度です。 |
| `alpha` | - | 散乱パラメータです。1に近い、または1を超える場合、単純な非集団的ガウス近似の妥当性に注意が必要です。 |
| `R^2` | - | フィッティングの決定係数です。1に近いほど、データをよく説明しています。 |

## 4. 入力に迷ったときの初期値

| 項目 | 初期値の目安 |
|---|---:|
| 波長校正 | `0.021` nm/pixel |
| レーザー波長 | `532` nm |
| 散乱角 | `90` deg |
| 装置幅sigma | `0` pixel |
| 気体温度 | `300` K |
| ラマン有効断面積 | `3.82e-34` m^2。ただし窒素Stokes線の論文値なので、水素ラマンでは条件に合う値へ変更してください。 |
| TS断面積 | `7.94e-30` m^2 |
| shot数 | 補正しない場合は `1` |
| Energy | 補正しない場合は `1` |
| 補正係数 | 補正しない場合は `1` |

## 5. 注意

- 波長ストップでラマンピークが欠けている場合、直接積分では強度を過小評価しやすいため、ガウス面積による推定を優先してください。
- 複数Stokesピークを合計する場合、入力するラマン有効断面積も合計ピークに対応した値にしてください。
- 現在の電子温度計算は、トムソン散乱スペクトルをガウス形で近似する試作実装です。集団的散乱が強い条件では、より厳密なスペクトル形状モデルが必要です。
- `R^2` が高くても、迷光・飽和・波長ストップ・背景処理の影響が残る場合があります。必ずフィット図も確認してください。
