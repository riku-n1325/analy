# LTSラマン校正トムソン散乱解析アプリ

このアプリは、ラマン散乱を用いた絶対値校正により、トムソン散乱スペクトルから電子密度と電子温度を求めるための試作版です。

参照式は、`Raster Thomson scattering in large-scale laser plasmas produced at high repetition rate` の Sec. III と Sec. IV に基づいています。

## 起動

`LTS_Analysis_App.app` フォルダ内の次のファイルをダブルクリックします。

```text
起動.bat
```

## ラマン校正TS解析タブ

タブ名は `ラマン校正TS解析` です。

入力欄では、数値入力枠の右側に単位を表示しています。論文中で実験値や引用値として与えられていた値も、固定値にはせず、ユーザーが変更できる入力欄にしています。

主な入力値は次の通りです。

- トムソンSPE: トムソン散乱スペクトルの`.spe`
- ラマンSPE: ラマン散乱スペクトルの`.spe`
- 水素圧力: チャンバー内ガス圧力 `Pa`
- 気体温度: ガス温度 `K`
- 波長校正: `nm/pixel`
- レーザー波長: `nm`
- 散乱角: `deg`
- ラマン有効断面積: 論文 Eq. (8) の Stokes 総和断面積 `m^2`
- TS断面積: 論文 Eq. (3) のトムソン散乱断面積 `m^2`
- ラマンshot数: ラマン信号の積算ショット数
- TS shot数: トムソン信号の積算ショット数
- ラマンEnergy: ラマン測定時のプローブエネルギー `mJ`
- TS Energy: トムソン測定時のプローブエネルギー `mJ`
- 補正係数: 透過率、ゲイン、レーザーエネルギー差などをまとめた補正係数

論文値に対応する初期値:

- ラマン有効断面積: `3.82e-34 m^2`
- TS断面積: `r_e^2 = 7.94e-30 m^2`
- 波長校正: ユーザー指定値 `0.021 nm/pixel`

## 使用している式

論文 Eq. (8):

```text
N_Stokes = k * n_gas * sigma_Stokes
```

ここから装置定数 `k` を求めます。

```text
k = N_Stokes / (n_gas * sigma_Stokes)
```

論文 Eq. (3):

```text
N_T = k * n_e * sigma_T
```

したがって電子密度は次で求めます。

```text
n_e = N_T / (k * sigma_T)
```

アプリ内では、ショット数で割った1ショットあたりの信号を使います。

```text
N_Stokes = Raman area / Raman shots
N_T      = Thomson area / Thomson shots
```

レーザーエネルギーを入力した場合、アプリ内では次のように `shot数 × probe energy` で規格化した信号を使います。

```text
N_Stokes = Raman area / (Raman shots * Raman energy)
N_T      = Thomson area / (Thomson shots * Thomson energy)
```

このとき装置定数 `k` は `m/mJ` 相当の値として扱われ、同じエネルギー規格化をしたトムソン信号に適用されます。

圧力から中性ガス密度を求めます。

```text
n_gas = P / (k_B * T_gas)
```

トムソン散乱幅から電子温度を求める式は、非集団的なガウス近似に基づくドップラー幅の式です。

```text
T_e[eV] = (m_e c^2 / e) * (sigma_lambda / (2 lambda_i sin(theta/2)))^2
```

また、論文 Sec. IV に従って散乱パラメータも出力します。

```text
alpha = 1 / (k_s * lambda_D)
k_s = 4 pi sin(theta/2) / lambda_i
```

`alpha` が大きい場合、スペクトルは単純なガウスではなく弱集団・集団的な形になり得ます。その場合は、論文 Eq. (9) のスペクトル密度関数によるフィットが必要になります。現在のアプリはまず非集団的ガウス近似とラマン絶対値校正を実装しています。

## 波長ストップで欠けたラマンピーク

ラマンピーク中心が波長ストップで欠けている場合は、`ストップ最小px` と `ストップ最大px` に遮光範囲を入力します。

その範囲をフィットから除外し、残った裾からガウス全体を外挿します。この場合、面積種別は `ガウス面積` を使ってください。`直接積分` は見えている部分だけの面積になるため、校正値を過小評価します。

## 複数Stokesピークの合計面積

回転ラマン散乱のStokes側に櫛状のピークが複数見えている場合は、面積種別で `複数Stokes合計` を選びます。
`Stokes探索最小` と `Stokes探索最大` に、合計したいStokesピーク群を含む pixel 範囲を入力してください。
アプリはその範囲内の局所ピークを検出し、各ピークをガウスフィットして、ガウス面積を合計します。

主な入力値:

- Stokes探索最小: 合計対象にするStokes側ピーク範囲の最小 pixel
- Stokes探索最大: 合計対象にするStokes側ピーク範囲の最大 pixel
- 最大ピーク数: 検出するピーク数の上限
- ピークしきい値: 最大ピーク高さに対する検出しきい値。小さくすると弱いピークも拾いやすくなります。

注意点として、ラマン有効断面積 `sigma_Stokes` は、ここで合計したピーク集合と同じ集合に対応する値を入力してください。
例えば、画像側で5本のStokesピーク面積を合計した場合、断面積もその5本分の和として扱う必要があります。
画像からピーク面積は求められますが、散乱断面積そのものは分子定数、温度、レーザー波長、偏光、文献値または理論モデルから別途与える必要があります。

## 測定日内ドリフトの時間補間

計測日の最初と最後でラマン散乱強度が変化する場合は、校正モードで `時間補間` を選びます。
このモードでは、開始ラマンSPEと終了ラマンSPEからそれぞれ装置定数を求め、トムソン測定時刻における装置定数を線形補間します。

```text
k_start = N_Stokes,start / (n_gas,start * sigma_Stokes)
k_end   = N_Stokes,end   / (n_gas,end   * sigma_Stokes)
k(t)    = k_start + (k_end - k_start) * (t - t_start) / (t_end - t_start)
n_e(t)  = N_T(t) / (k(t) * sigma_T)
```

入力値:

- 開始ラマンSPE: 計測日の最初またはTS測定前に取得したラマンSPE
- 終了ラマンSPE: 計測日の最後またはTS測定後に取得したラマンSPE
- 開始時刻、終了時刻、TS時刻: 同じ基準からの経過時間 `h`
- 開始圧力、終了圧力: 各ラマン測定時のガス圧力 `Pa`
- 開始shot数、終了shot数: 各ラマン測定の積算ショット数 `shots`
- 開始Energy、終了Energy: 各ラマン測定時のプローブエネルギー `mJ`

同じ圧力、同じshot数、同じプローブエネルギーで測っている場合でも、これらの値を入れておくと解析条件がCSVに残るため、後から確認しやすくなります。

## 出力

解析ボタンを押すと、ログ欄に次を表示します。

- トムソン中心
- トムソンFWHM
- トムソンR^2
- ラマン中心
- ラマンFWHM
- ラマンR^2
- 中性ガス密度 `n_gas`
- 装置定数 `k`
- 電子密度 `n_e`
- 電子温度 `T_e`
- 散乱パラメータ `alpha`

同じ内容を `latest_lts_result.csv` に保存します。

## ラマン圧力依存性タブ

ラマン散乱スペクトル強度の圧力依存性を確認するためのタブです。

使い方は次の通りです。

1. `フォルダ`で、圧力依存性を確認したい`.spe`ファイル群が入ったフォルダを選択します。
2. `SPE一覧読込`を押すと、フォルダ内の`.spe`ファイルが一覧表示されます。
3. 各ファイルの右側に、そのスペクトルを測定した圧力を入力します。単位は `Pa` です。
4. 必要に応じて、`ラマン中心`、`ストップ最小`、`ストップ最大`を入力します。単位は `pixel` です。
5. 縦軸を `ピーク高さ` または `ガウス面積` から選びます。
6. `解析`を押すと、横軸圧力、縦軸ラマン強度のグラフを表示します。

出力される値:

- 圧力 `Pa`
- ピーク高さ `counts` またはガウス面積 `count*pixel`
- ガウスフィット中心 `pixel`
- FWHM `pixel`
- フィット評価値 `R^2`

解析結果は、選択したフォルダ内に次のCSVとして保存されます。

```text
raman_pressure_dependence.csv
```

## 注意

現在の電子温度計算は非集団的ガウス近似です。論文 Eq. (9) の弱集団・集団的スペクトル密度関数フィットは、今後追加する拡張候補です。
